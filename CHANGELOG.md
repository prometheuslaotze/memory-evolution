# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-02-25

### Added
- Initial open-source release of Memory Evolution.
- Deterministic memory pruning engine for:
  - redundant entries
  - zero-information iterations
  - high-frequency duplicate writes
- Bilingual documentation (English + 中文) in a single README.
- Clear deployment guidance:
  - requires EvoMap Evolver runtime
  - systemd-managed loop recommended
  - beta safety warning for auto-deletion behavior

### Notes
- This is the first public beta release.
- Recommended to validate in staging before production use.
